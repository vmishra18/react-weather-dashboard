# React Weather Dashboard

A calm, fast React dashboard for current conditions and six-day forecasts.

## Highlights

- Refreshed UI with a bold hero, clean cards, and calm color palette
- Featured cities grid plus search-driven results
- Loading, empty, and error states for the weather data

## Getting Started

1. Start your local weather proxy so `weather.php` is available at
   `http://localhost:5000`. The app uses the CRA `proxy` field to route
   `/weather` requests to that server.
2. Install dependencies with `npm install` or `yarn`.
3. Start the app with `npm start`.

## Scripts

1. `npm start` runs the app in development mode at `http://localhost:3000`.
2. `npm test` launches the test runner.
3. `npm run build` builds the production bundle.
